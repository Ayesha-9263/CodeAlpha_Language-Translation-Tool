## 🚀 Project Overview

In this project, I built a simple and interactive Language Translation Tool using Python.  
The main goal of this project is to allow users to translate text from one language to another using an easy command-line interface.

This tool uses an external translation library to process user input and return accurate translated output in real time.

---

## ✨ Features

- 🌐 Supports multiple languages (like English, Hindi, French, etc.)
- 💬 User-friendly input system
- 🔁 Continuous translation option (loop-based execution)
- ⚡ Fast and efficient translation using API
- 🧠 Beginner-friendly and easy to understand code

---

## 🛠️ Technologies Used

- Python
- deep-translator library

---

## 📌 How It Works

1. The user enters the text they want to translate  
2. Selects the source language  
3. Selects the target language  
4. The program processes the input using the translator  
5. Displays the translated text instantly  

The program continues running until the user decides to exit.
